# 📰 The Voices

**The Voices** is a premium blogging platform designed with a bold, brutalist newspaper aesthetic. Built for writers and readers who appreciate high-impact design and a seamless writing experience.

## ✨ Features

- 🖋️ **Rich-Text Editor**: Powered by [Tiptap](https://tiptap.dev/), offering a clean and distraction-free writing environment.
- 🗞️ **Brutalist Design**: A unique "newspaper" inspired UI featuring heavy borders, vintage typography, and modern micro-interactions.
- 🔐 **Secure Authentication**: Built-in authentication system using Firebase Auth.
- 🚀 **Real-time Database**: Lightning-fast data syncing with Firebase Firestore.
- 📊 **Admin Dashboard**: Full-featured administrative panel for content moderation and site management.
- 🔍 **Advanced Search**: Easily find articles by keywords, authors, or categories.
- 📱 **Fully Responsive**: Optimized for every screen size, from mobile devices to desktop monitors.
- 🎭 **Smooth Animations**: Interactive elements enhanced with Framer Motion.

---

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Backend/Auth**: [Firebase](https://firebase.google.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Editor**: [Tiptap](https://tiptap.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Firebase project

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/the-voices.git
   cd the-voices
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Configuration:**
   Create a `.env` file in the root directory and add your Firebase configuration:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   ```

4. **Seeding the Database (Optional):**
   To populate your database with initial vintage-style articles:
   ```bash
   npm run seed
   ```

5. **Run the development server:**
   ```bash
   npm run dev
   ```
